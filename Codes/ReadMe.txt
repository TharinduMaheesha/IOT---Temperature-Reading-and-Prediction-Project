This Folder contains all the codes used for this Project

1) main.py - The main file when executed handles the sensor along with the actuator

2) Time_series_model.ipynb - This file contains the code which was used to develop the time series model

3) Time_series_prediction.ipynb - This file contains the code used to handle the predictions based on the built model

4) prediction_visualization.js & past_data_function.js - Contains the js code used to adjust the result set read from the database

5) json files - contains the json code for the flows bult on node-red
